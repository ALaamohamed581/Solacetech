 
---

### Frontend
To run this app:
```bash
cd frontend
npm i 
npm run dev
```

### Backend
To run this app:
```bash
cd ..
cd backend
npm i 
npm run start:dev
```

### .env Backend Variables
```bash
MONGO_URL=<your_mongo_url>
PORT=<your_backend_port>
```

### .env Frontend Variables
```bash
VITE_BASE_URL=<your_base_url>
```
